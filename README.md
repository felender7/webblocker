## Getting Started

Block any website based on time and current date.


#### Requirements

You'll need the following installed to run the application successfully:
* `Python 3`


#### Windows Set up

* `run python3 app.pyw `

### Mac & Linux Set up
* run the following commands
* `sudo app.py`
* `sudo crontab -e`
* `@reboot python3 /home/user/application/app.py` [your path]
* `restart the machine`
